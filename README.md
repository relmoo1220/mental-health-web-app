<!-- GETTING STARTED -->
## Getting Started

### Prerequisites
* Flask
* Docker

### Installation

1. Clone the repo
  ```sh
  git clone https://github.com/relmoo1220/psd2.git
  ```

### Flask
2. Run the Flask server
  ```bash
  python app.py
  ```

### Docker
3. Build docker image in backend PATH
  ```bash
  docker build -t mental .
  ```
4. Run docker image **IN EC2** (development)
  ```bash
  docker run mental:latest
  ```
