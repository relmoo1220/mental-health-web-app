## Getting Started

### Prerequisites
* Flask
* Docker

### Installation

1. Clone the repo
  ```sh
  git clone https://github.com/relmoo1220/psd2.git
  ```

### Flask
2. Run the Flask server
  ```bash
  python app.py
  ```

### Docker
3. Build and run docker image
  ```bash
  docker compose up
  ```
